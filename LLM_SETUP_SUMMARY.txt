⚠️  IMPORTANT NOTES
════════════════════════════════════════════════════════════════════════════

1. API Key Security
   • Never commit API keys to git
   • Use environment variables or .env files (in .gitignore)
   • Rotate keys regularly

2. Rate Limits
   • Gemini free tier: 15 requests per minute
   • Wait 1 minute if you hit the limit
   • Use paid tier for production workloads

3. GPU Configuration
   • Device: cuda (already set in config)
   • Encoder: MedCPT on GPU (set in previous fix)
   • LLM: CPU inference only (standard)

4. Fallback Strategy
   • If Gemini fails, can fall back to OpenAI
   • Both clients share same interface
   • Configure in pipeline_config.yaml

5. Model Updates
   • Gemini models may be updated by Google
   • Check available models: https://ai.google.dev/models
   • Update model name in config if needed
